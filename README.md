# healthcare-leads-flow-dashboard

                                     ## Project Overview ##

This project is an interactive healthcare analytics dashboard developed using Power BI to analyze hospital lead flow, patient trends, billing analysis, diagnosis patterns, and doctor feedback performance.

The dashboard helps visualize patient admissions, discharge trends, hospital billing insights, diagnosis distribution, and healthcare operational performance.

---

## Dashboard Features

### Patient Flow Analysis

* Admit Date Tracking
* Discharge Date Tracking
* Follow-up Date Monitoring

### Billing Analysis

* Total Bill Amount Analysis
* Health Insurance vs Billing Comparison
* Test-wise Billing Insights

### Diagnosis Analysis

* Diagnosis-wise Patient Count
* Disease Distribution Analysis
* Patient Trend Visualization

### Hospital Operations Analysis

* Bed Occupancy Tracking
* ICU vs General vs Private Bed Analysis

### Doctor Feedback Analysis

* Feedback Volume per Doctor
* Doctor Performance Insights
